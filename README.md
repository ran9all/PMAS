# PMAS — Clinical Planning System

**Планирование и анализ лицевой хирургии на базе AI**

> Веб-приложение для клинического анализа лица, планирования пластических операций и документирования результатов. Работает в браузере — без установки.

---

## Запуск

### Онлайн
https://ran9all.github.io/PMAS/

### Локально
- **macOS** — двойной клик на `start.command`
- **Windows** — двойной клик на `start.bat`

> Требуется Python 3 и современный браузер (Chrome, Edge, Firefox).

## Инструкция: 2D Анализ

### 1. Загрузка фото
- Откройте вкладку **«2D Анализ»**
- Нажмите **«Загрузить фото»** и выберите фронтальный снимок лица (JPG/PNG)
- Фото появится на рабочей области

### 2. AI-анализ
- Нажмите **«AI ориентиры»** — система автоматически найдёт ключевые точки лица
- На фото появятся: средняя линия, горизонталь глаз, три трети лица
- В панели отобразятся: индекс гармонии, симметрия, пропорции, отклонение носа

### 3. Калибровка
- Нажмите **«Калибровка»**
- Укажите две точки на фото с известным расстоянием между ними
- Введите реальное расстояние в мм — все дальнейшие измерения будут в миллиметрах

### 4. Ручные инструменты
- **Расстояние** — кликните 2 точки → покажет расстояние (H, L и AB = √(H² + L²))
- **Вектор** — кликните 2 точки → стрелка направления перемещения тканей
- **Наклон** — кликните 2 точки → угол наклона относительно горизонта
- **Угол** — кликните 3 точки → клинический угол между отрезками
- **Зона** — обведите область → полигон с расчётом площади

### 5. Готовые пресеты
Нажмите **«Пресеты»** и выберите один из готовых наборов:
- **Кантинг глаз** — наклон линии глаз
- **Ось носа** — наклон от glabella до кончика носа
- **Носогубный угол** — 3-точечный угол (columella → subnasale → верхняя губа)
- **Линия подбородка** — наклон нижней трети
- **Фейслифтинг** — базовые векторы, SMAS plication, Deep-plane, MACS

### 6. Асимметрия
- Система автоматически сравнивает парные элементы (R vs L)
- Показывает разницу в длине, угле, площади и смещении

### 7. До / После
- Нажмите **«Сохранить До»** — запомнит текущее состояние
- Внесите изменения в план
- Нажмите **«Показать До»** — наложит сохранённый снимок для сравнения

### 8. Экспорт
- **PDF** — нажмите кнопку «PDF» → скачается отчёт со скриншотом и всеми измерениями
- **DOCX** — нажмите кнопку «DOCX» → скачается документ для медкарты

## Инструкция: 3D Модель

### 1. Выбор модели
- Откройте вкладку **«3D Модель»**
- Выберите модель из списка (Lee Perry-Smith или Face Cap)
- Или загрузите свой файл `.glb` / `.gltf` кнопкой **«Загрузить»**

### 2. Навигация
- **ЛКМ + перетаскивание** — вращение модели
- **ПКМ + перетаскивание** — перемещение
- **Колесо мыши** — масштаб
- **Сброс вида** — вернуть камеру в исходное положение

### 3. Режимы отображения
- **Wireframe** — каркасный вид (сетка полигонов)
- **Нормали** — цветная карта направлений поверхности
- **Свет**: Дневной / Мягкий / Студия — три варианта освещения

### 4. Калибровка
- Нажмите **«Калибровка»**
- Кликните 2 точки на модели с известным расстоянием
- Введите реальное расстояние в мм

### 5. Инструменты (кликайте прямо по поверхности модели)
- **Точка** — клик на модель → ставит маркер с координатами
- **Расстояние** — 2 клика → линия + расстояние в мм
- **Вектор** — 2 клика → стрелка направления + длина
- **Угол** — 3 клика → угол между двумя отрезками в градусах
- **Наклон** — 2 клика → угол наклона относительно горизонтальной плоскости
- **Измерение** — 2 клика → линия + расстояние (универсальная линейка)

### 6. План операции
- Все измерения автоматически добавляются в список **«Элементы плана»** слева
- Каждый элемент можно удалить кнопкой **«Удалить»**
- Кнопка **«Очистить всё»** — сбросить все аннотации

### 7. Экспорт
- **PDF** — скриншот 3D-модели с аннотациями + список измерений
- **DOCX** — структурированный документ с данными
